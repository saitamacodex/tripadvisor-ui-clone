## Tripadvisor clone

## 📌 Project Goal

The goal of this project is to:

- Practice **real-world website cloning**
- Improve **HTML5 semantic structure**
- Learn **modern CSS layout techniques**
- Build **responsive layouts using pure CSS**
- Avoid CSS frameworks to understand **core CSS fundamentals**

## 🛠️ Built With

- **HTML5**
- **CSS3**
- **Semantic HTML**
- **Flexbox**
- **CSS Grid**
- **Responsive Design**
